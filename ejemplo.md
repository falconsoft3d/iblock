# Bienvenido a IBlock

¡Hola! Este es tu nuevo bloc de notas estilo Windows creado con Flutter.

## Características principales:

✅ **Archivo**
- Nuevo documento (Ctrl+N)
- Abrir archivo (Ctrl+O)
- Guardar (Ctrl+S)
- Guardar como (Ctrl+Shift+S)

✅ **Edición**
- Seleccionar todo (Ctrl+A)
- Cortar (Ctrl+X)
- Copiar (Ctrl+C)
- Pegar (Ctrl+V)

✅ **Formato**
- Ajuste de línea activable

✅ **Información**
- Contador de líneas y columnas
- Indicador de cambios no guardados
- Barra de estado

## Formatos soportados:
- .txt (archivos de texto)
- .md (Markdown)
- .dart (código Dart)
- .json (archivos JSON)

## Compatibilidad:
- ✅ macOS (10.14+)
- ✅ Windows (7+)

---

¡Prueba abrir este archivo para ver cómo funciona la aplicación!

Para comenzar, puedes:
1. Crear un nuevo documento con Ctrl+N
2. Escribir tu texto
3. Guardarlo con Ctrl+S
